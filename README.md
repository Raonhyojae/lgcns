# 🏢 LG CNS 통합 물품관리 시스템

LG CNS의 물품 회수 및 인수 업무를 위한 웹 기반 통합 관리 시스템입니다.

## ✨ 주요 기능

### 📋 이중 모드 지원
- **회수확인서 모드**: 물품 회수 시 사용하는 확인서 생성
- **인수확인서 모드**: 물품 인수 시 사용하는 확인서 생성

### 📊 엑셀 파일 처리
- `.xlsx`, `.xls` 파일 업로드 지원
- 파일명 기반 자동 모드 검증
- 실시간 진행률 표시
- 최대 50MB 파일 지원

### 🖊️ 디지털 서명
- 터치/마우스 기반 서명 입력
- 사용자/대리인 서명 분리 관리
- 로컬 스토리지 기반 서명 저장
- 모바일 최적화된 서명 캔버스

### 📷 바코드 스캔
- 카메라 기반 바코드/QR코드 스캔
- 블루투스 바코드 스캐너 지원
- 시리얼번호 자동 입력
- 권한 관리 및 에러 처리

### 🖨️ 인쇄 최적화
- A4 용지 기준 2개씩 인쇄
- 절취선 포함 레이아웃
- 인쇄 전용 스타일 적용

## 🛠️ 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업
- **CSS3**: 반응형 디자인, 그라디언트, 애니메이션
- **JavaScript (ES6+)**: 모듈화된 코드 구조

### 라이브러리
- **XLSX.js**: 엑셀 파일 파싱
- **ZXing**: 바코드/QR코드 스캔
- **DOMPurify**: XSS 보안 처리

### 보안 기능
- 입력 데이터 새니타이징
- 파일 타입/크기 검증
- 권한 기반 카메라 접근

## 🚀 사용 방법

### 1. 파일 업로드
```
1. 모드 선택 (회수확인서/인수확인서)
2. 엑셀 파일 업로드
3. 파일명 검증 통과 확인
```

### 2. 확인서 생성
```
1. "확인서 생성" 버튼 클릭
2. 자동 데이터 파싱 및 그룹핑
3. 확인서 레이아웃 생성
```

### 3. 서명 입력
```
1. 서명 박스 클릭
2. 디지털 서명 입력
3. 저장 및 적용
```

### 4. 바코드 스캔 (선택사항)
```
1. "스캔" 버튼 클릭
2. 카메라 권한 허용
3. 바코드 스캔 후 자동 입력
```

### 5. 인쇄
```
1. "인쇄하기" 버튼 클릭
2. A4 용지 설정 확인
3. 인쇄 실행
```

## 📋 파일 규칙

### 회수확인서 모드
- **허용 키워드**: 회수, 회수리스트, 회수목록, 반납
- **차단 키워드**: 납품, 자산납품, 인수, 배송, 전달

### 인수확인서 모드
- **허용 키워드**: 납품, 자산납품, 인수, 배송, 전달, 인수리스트
- **차단 키워드**: 회수, 회수리스트, 반납

## 🗂️ 엑셀 파일 구조

### 회수확인서용 엑셀
```
A열: 자산번호
B열: 회수코드
C열: 자산유형
D열: 모델명
J열: 담당자명
K열: 연락처
```

### 인수확인서용 엑셀
```
A열: 계약번호
B열: 제품명
C열: 수량
H열: 검수자
```

## 🔧 설치 및 실행

### 로컬 실행
```bash
# 파일 다운로드
git clone [repository-url]

# 웹 서버 실행 (예: Python)
python -m http.server 8000

# 브라우저에서 접속
http://localhost:8000
```

- 정적 웹 호스팅 서비스 사용 가능

## 🐛 문제 해결

### 카메라 권한 오류
```
1. 브라우저 주소창의 자물쇠 아이콘 클릭
2. 카메라 권한을 "허용"으로 변경
3. 페이지 새로고침
```

### 파일 업로드 실패
```
1. 파일 크기 확인 (최대 50MB)
2. 파일 형식 확인 (.xlsx, .xls)
3. 파일명 키워드 확인
```

### 인쇄 문제
```
1. A4 용지 설정 확인
2. 여백 최소화 설정
3. 배경 그래픽 인쇄 활성화
```
